# vue-project

#### 介绍
基于webpack的vue项目
+ mint-ui
  + Lazyload 图片懒加载 http://mint-ui.github.io/docs/#/zh-cn2/lazyload
+ vue-preview 预览图片 https://github.com/LS1231/vue-preview
+ moment 时间格式化 https://momentjs.com/docs/#/use-it/

#### 使用说明
1.  安装依赖
```npm install```
2.  运行
```npm run dev```
